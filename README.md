# 🧬 RepoDNA

> Reveal the DNA of any GitHub repository.

RepoDNA is a GitHub repository analysis platform that helps developers quickly understand any public repository by analyzing its structure, technologies, dependencies, programming languages, statistics, README, and recent commit activity—all from a single dashboard.

## ✨ Features

- 🔍 Analyze any public GitHub repository
- 📊 Repository overview and metadata
- 🌳 Interactive project structure visualization
- 💻 Technology stack detection
- 📦 Dependency analysis
- 🌐 Programming language distribution
- 📈 Repository statistics
- 📖 README viewer
- 📝 Recent commit history
- ⚡ Fast and responsive dashboard
- 🎨 Modern UI with smooth animations
